# PopX Assignment 🚀

A modern React application built with Vite and Tailwind CSS.

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM

## 📱 Key Features

- Responsive Design
- Modern UI/UX
- Authentication Flow
- User Settings Panel

## 📂 Project Structure

```
src/
├── pages/
│   ├── Welcome.jsx
│   ├── Signup.jsx
│   ├── Signin.jsx

│   └── SettingsPanel.jsx
├── components/
├── assets/
└── styles/
```

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/SanojS31/educase-assignment.git
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```
